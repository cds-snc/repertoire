# Ordered List

The Ordered List component is used to build styled HTML numbered lists (`<ol>`). This can wrap `<li>` tags or List Item components.

<Playground>
  <OrderedList>
    <li>Ordered list</li>
  </OrderedList>
  <OrderedList color="red" fontWeight="bold">
    <li>red, bold ordered list</li>
  </OrderedList>
  <OrderedList fontSize="20px">
    <li>Large ordered list</li>
  </OrderedList>
</Playground>

## Presets

`<Ol />` is a preset List Item configurition for a basic HTML ol. listStyleType can be passed to set different bullet types such as roman numerals and none.
Passing none to list style will result in a list with no left margin or numbers.

<Playground>
  <Ol>
    <li>Ordered list preset list item</li>
  </Ol>
  <Ol listStyleType="upper-roman">
    <li>Ordered list with upper roman numerals</li>
  </Ol>
  <Ol listStyleType="none">
    <li>Ordered list without numbers</li>
  </Ol>

</Playground>

## styled-system props

Component accepts [space](https://styled-system.com/api#space), [color](https://styled-system.com/api#color),
[fontSize](https://styled-system.com/api#fontsize), [typography](https://styled-system.com/api#typography), [width](https://styled-system.com/api#width)
and [layout](https://styled-system.com/api#layout) props from styled-system.

## Component props

| Name          |  Type  | Default  | Description                          |
| :------------ | :----: | -------- | :----------------------------------- |
| listStyleProp | String | `number` | Sets the list-style-type in the css. |
