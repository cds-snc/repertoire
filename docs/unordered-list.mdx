# Unordered List

The Unordered List component is used to build styled HTML bulleted lists (`<ul>`). This can wrap `<li>` tags or List Item components.

<Playground>
  <UnorderedList>
    <li>Unordered list</li>
  </UnorderedList>
  <UnorderedList color="red" fontWeight="bold">
    <li>red, bold unordered list</li>
  </UnorderedList>
  <UnorderedList fontSize="20px">
    <li>Large unordered list</li>
  </UnorderedList>
</Playground>

## Presets

`<Ul />` is a preset List Item configurition for a basic HTML ul. listStyleType can be passed to set different bullet types such as disc, square and none.
Passing none to list style will result in a list with no left margin or bullets.

<Playground>
  <Ul>
    <li>Unordered list preset list item</li>
  </Ul>
  <Ul listStyleType="square">
    <li>Unordered list with square bullets</li>
  </Ul>
  <Ul listStyleType="none">
    <li>Unordred list without bullets</li>
  </Ul>

</Playground>

## styled-system props

Text component accepts [space](https://styled-system.com/api#space), [color](https://styled-system.com/api#color),
[fontSize](https://styled-system.com/api#fontsize), [typography](https://styled-system.com/api#typography), [width](https://styled-system.com/api#width)
and [layout](https://styled-system.com/api#layout) props from styled-system.

## Component props

| Name          |  Type  | Default | Description                          |
| :------------ | :----: | ------- | :----------------------------------- |
| listStyleProp | String | `disc`  | Sets the list-style-type in the css. |
